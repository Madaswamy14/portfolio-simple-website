# Madaswamy's Portfolio

A personal portfolio website showcasing the work, skills, and interests of Madaswamy, a Web Developer.

## 📁 Project Structure

```
Portfolio/
├── index.html           # Main portfolio page
├── contact_form.html    # Contact form page
├── style.css            # Styling for the portfolio
└── README.md            # This file
```

## 📄 Files Overview

### `index.html`
The main landing page of the portfolio featuring:
- **Section 1**: Introduction with name, professional role (Web Developer), and contact button
- **Section 2**: About Me - Personal bio and background
- **Section 3**: My Interests - List of personal interests including web development, reading, and photography
- **Section 4**: My Projects - Showcase of notable projects (e.g., Amazon E-commerce Clone, Travel Website)
- **Section 5**: Social Media Links - Links to external social media profiles

### `contact_form.html`
A dedicated contact form page for visitors to get in touch with the portfolio owner.

### `style.css`
Comprehensive styling for the portfolio including:
- Modern typography using "Young Serif" Google Font
- Responsive section layouts with semantic CSS class naming (BEM methodology)
- Black and white color scheme with orange-red accent colors
- Styled contact buttons and social media links
- Professional spacing and padding throughout

## 🎨 Design Features

- **Clean Typography**: Uses "Young Serif" font for elegant professional appearance
- **Section-Based Layout**: Organized into 5 distinct sections for easy navigation
- **Color Scheme**: 
  - Primary: Black and White
  - Accent: Orange-red for call-to-action buttons
  - Links: Sky blue for social media
- **BEM Naming Convention**: CSS classes follow Block Element Modifier pattern for maintainability

## 🚀 Getting Started

1. Open `index.html` in your web browser to view the portfolio
2. Click the "Contact" button to navigate to the contact form
3. Explore the portfolio sections to learn more about Madaswamy's work and interests

## 📝 Customization

To customize this portfolio:
- Update personal information in `index.html`
- Add or remove project entries in the "My Projects" section
- Modify colors in `style.css` by changing the color values
- Update social media links in the footer section
- Modify fonts and typography in the CSS file
